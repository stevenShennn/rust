## Cargo 常用命令

- cargo new project_name
- cargo build
- cargo run
- cargo check
- cargo build --release : 优化编译
- cargo update : 更新依赖
- cargo doc --open : 生成文档并打开
- cargo clean : 清除编译文件
- cargo fmt : 格式化代码
- cargo clippy : 检查代码风格
- cargo test : 运行测试
- cargo bench : 运行基准测试
- cargo publish : 发布包
